LaTeX Online

you give it a link, it gives you PDF

Example: /compile?git=https://github.com/posquit0/Awesome-CV&target=examples/resume.tex&command=xelatex

Capabilites

  1. Build a file by a link Tell me more
  2. Build a public git repo Tell me more
  3. Build your any local tex files Tell me more

Parameters

You can specify latex command and target file as in the example above.

  1. &command= xelatex or pdflatex
  2. &target= define target file in GIT repository

What is going to happen

  1. The file is fetched and saved with .tex extension
  2. Hash sum of the file is counted
  3. If we have the compiled pdf for the hash sum in our cache, then we return cached PDF
  4. Otherwise we compile the file using pdflatex , store the result in cash and return it
